# Minutes

Transcripts of Joint Expeditions, filed as provenance for the entries
distilled from them. One file per meeting, named
`NNNN-<headword>.md`, where `NNNN` matches the session number in
`SOCIETY_LOG.md`.

Minutes are committed alongside the entry the Publisher distills from
them. Adjourned meetings (those producing no entry) commit minutes
anyway — failure is also archive.

Minutes are **not edited after committing**. If a sentence in the
minutes looks wrong in hindsight, that is what the next dissent is
for.

See `MEETINGS.md` for the full Joint Expedition procedure.
